# Baofeng BF-T17

**Frequency Range** UHF 400-470MHz
**Operation Voltage** DC 3.7 V ±10%
**Operation Mode** Simplex
**Dimensions** 147mm×54mm×27mm (with Antenna)
**Weight** 130g (Battery pack included)
**RF Output Power** 2W (Max)
